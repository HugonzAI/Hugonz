## 交互状态

### 按钮状态
```
Normal   → 默认渐变色
Hover    → 更深的渐变色
Pressed  → 纯色（最深色）
Disabled → 灰色（#d1d5db）
```

### 连接状态指示
```
● 灰色   = 未连接
● 绿色   = 已连接
● 红色   = 连接失败（未使用）
```
